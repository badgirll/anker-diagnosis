# Ankerのタイプ診断

**🔗 デモサイト**: https://anker-diagnosis.vercel.app/

Anker製品の購入を検討しているユーザー向けの診断ツールです。4つの質問に答えることで、あなたのライフスタイルに合った充電器・モバイルバッテリーをレコメンドします。

![Ankerのタイプ診断](https://anker-diagnosis.vercel.app/ogp.png)

## ✨ 特徴

- 📱 **レスポンシブデザイン** - PC・スマホで最適な表示
- ⚡ **4つの診断タイプ** - あなたに最適なAnker製品を提案
- 🎨 **Ankerブランドデザイン** - 公式サイトのカラー・トンマナに準拠
- 📤 **シェア機能** - 診断結果を画像でダウンロード・SNSシェア
- 🖼️ **OGP対応** - SNSでシェアした時に画像が表示される

## 🎯 診断結果タイプ

| タイプ | 特徴 |
|:---|:---|
| **スピードスター** | 効率派でシゴデキ！時間を無駄にしたくない、急速充電重視 |
| **ミニマリスト** | 手ぶら至上主義！小型・軽量・スマートなデザイン好き |
| **面倒くさがりさん・心配性** | 忘れ物が不安！ケーブル一体型・充電器とバッテリー一体型が安心 |
| **本物志向** | 妥協したくない！最高スペック・多ポート・大容量重視 |

詳細な設計仕様・質問内容・スコアリングロジックは **[診断設計書.md](./診断設計書.md)** を参照してください。

## 🛠️ 技術スタック

- **Next.js 16.1.6** - React フレームワーク (App Router)
- **TypeScript** - 型安全な開発
- **Tailwind CSS 4.x** - ユーティリティファーストCSS
- **Vercel** - デプロイ & ホスティング（自動デプロイ対応）
- **Web Share API** - スマホで写真に直接保存可能

## 🚀 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 📦 ビルド

```bash
# プロダクションビルド
npm run build

# ビルドしたアプリの起動
npm start
```

## 🎨 主要機能

### レスポンシブ対応
- **画像切り替え**: PC (769px以上) / スマホ (768px以下) で異なる画像を表示
- **レイアウト調整**: スマホはファーストビューに収まるよう上寄せ表示

### シェア機能
- **PC**: 診断結果画像をダウンロード
- **スマホ**: Share API経由で共有メニューを表示（写真に保存可能）
- **X (Twitter)**: ハッシュタグ付きでポスト

### デザイン
- Ankerブランドカラー (`#00a9e0`)
- Noto Sans JP フォント
- ホバーエフェクト・トランジション
